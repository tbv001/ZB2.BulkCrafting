# Bulk Crafting

**Bulk Crafting** is a mod for **Zumbi Blocks 2** that adds bulk crafting to the crafting system, allowing you to craft multiple items at once.

# Features

- Adds few buttons to adjust the craft amount in the crafting menu.
- Displays the current amount and maximum craftable quantity on the craft button.
- Automatically calculates the maximum craftable amount based on available materials.
- Updates material requirements in real-time as you adjust the amount.

## Installation

1. Make sure you have the latest version of **BepInEx 5** installed.
2. Download the latest version of the mod from the [releases](https://github.com/tbv001/ZB2.BulkCrafting/releases) page.
3. Copy the DLL into the `plugins` folder of your BepInEx installation:

```
Zumbi Blocks 2 Open Alpha\BepInEx\plugins\BulkCrafting.dll
```
